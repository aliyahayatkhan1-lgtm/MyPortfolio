# 🌐 Personal Portfolio Website

A modern, responsive, and interactive personal portfolio website showcasing my skills, projects, certifications, achievements, education, and professional experience in Artificial Intelligence, Machine Learning, Python Development, and Web Development.

## ✨ Features

* Responsive design for desktop, tablet, and mobile devices
* Modern dark/light mode toggle
* Smooth scrolling and scroll reveal animations
* Interactive project showcase with category filters
* Professional skills section with progress indicators
* Education and experience timeline
* Certifications and achievements gallery
* Resume preview and download option
* Contact form
* Animated statistics counters
* Particle background effects
* Typing animation on the hero section
* Back-to-top button
* Premium UI animations and hover effects

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Font Awesome
* Google Fonts
* Typed.js
* Particles.js

## 📁 Project Structure

```text
portfolio/

│── script.js
│── assets/
│   ├── images/
│   ├── certificates/
│   ├── projects/

```

## 🚀 Getting Started

1. Clone this repository.
2. Open the project folder.
3. Open `index.html` in your browser.

No additional installation is required.
